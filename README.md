# AI Competitor Intel Scraper 📊

A lightweight, high-speed Python module designed to fetch web data from any competitor landing page and instantly transform it into a structured business intelligence report using the Gemini LLM.

## ⚙️ Quick Installation

```bash
# Clone the repository
git clone [https://github.com/Bharath-Al-Modules/ai-competitor-intel.git](https://github.com/Bharath-Al-Modules/ai-competitor-intel.git)

# Move into the folder
cd ai-competitor-intel

# Install required packages
pip install -r requirements.txt
